# 获取API凭证指南

## 第一步：获取 Spoonacular API Key

### 1. 访问注册页面
打开浏览器，访问：https://spoonacular.com/food-api/console

### 2. 注册账户
- 点击 "Sign Up" 或 "Get API Key"
- 填写基本信息：
  - Email: 你的邮箱
  - Password: 设置密码
  - Name: 你的姓名或项目名称
- 勾选同意条款
- 点击 "Create Account"

### 3. 验证邮箱
- 检查邮箱，点击验证链接
- 完成邮箱验证

### 4. 获取API Key
- 登录后，进入 Dashboard
- 在 "API Keys" 或 "Console" 页面找到你的API Key
- 复制保存这个Key（格式类似：`abc123def456...`）

### 5. 了解免费额度
- **免费额度**：50 requests/day
- **付费升级**：$29/月 可获得1500 requests/day
- **当前足够**：我们的应用每天请求量不会超过免费额度

## 第二步：注册微信小程序

### 1. 访问微信公众平台
打开浏览器，访问：https://mp.weixin.qq.com

### 2. 注册小程序
- 点击右上角 "立即注册"
- 选择 "小程序"
- 使用微信扫码登录

### 3. 填写基本信息
- **账号类型**：个人/企业/政府等（选择个人即可）
- **邮箱**：用于接收审核通知
- **主体信息**：个人开发者信息
- **管理员信息**：你的个人信息

### 4. 选择服务类目
- **一级类目**：生活服务
- **二级类目**：美食
- **具体服务**：菜谱、美食推荐等

### 5. 提交审核
- 上传身份证照片
- 填写小程序信息：
  - **名称**：美食助手（或你喜欢的名字）
  - **头像**：美食相关图片
  - **介绍**：智能菜谱搜索和菜单规划工具
- 提交审核（通常1-2天通过）

### 6. 获取AppID和AppSecret

审核通过后：
1. 登录微信公众平台
2. 进入 "开发" -> "开发管理" -> "开发设置"
3. 找到 "AppID" 和 "AppSecret"
4. **AppID**：格式如 `wx1234567890abcdef`
5. **AppSecret**：格式如 `abc123def456...`

## 第三步：记录凭证信息

创建一个文本文件保存以下信息：

```
Spoonacular API Key: [你的API Key]
WeChat AppID: [你的AppID]
WeChat AppSecret: [你的AppSecret]
```

## 注意事项

### Spoonacular API
- API Key 是免费的，每天50次请求足够测试
- 如果需要更多请求，可以随时升级付费计划
- API 支持菜谱搜索、营养分析、食材信息等

### 微信小程序
- 个人开发者可以注册小程序
- 审核时间通常1-2个工作日
- AppID 和 AppSecret 很重要，不要泄露给他人
- 小程序注册需要实名认证

## 常见问题

**Q: Spoonacular注册失败？**
A: 确保使用有效的邮箱，密码不少于8位

**Q: 微信小程序审核被拒？**
A: 确保信息填写完整，头像和介绍符合要求

**Q: API Key显示无效？**
A: 检查是否复制完整，可能有空格或隐藏字符

**Q: 需要付费才能使用？**
A: Spoonacular有免费额度，微信小程序注册免费
