# 国内可用的菜谱API替代方案

## 推荐方案：聚合数据API

### 1. 注册账户
访问：https://www.juhe.cn/
- 注册账户（免费）
- 实名认证（需要身份证）

### 2. 获取菜谱API
- 搜索 "菜谱" 或 "美食"
- 选择 "菜谱大全" 或 "美食菜谱" API
- 申请免费试用

### 3. API信息
- **接口地址**: http://apis.juhe.cn/cook/
- **请求方式**: GET
- **参数**:
  - key: 你的API Key
  - menu: 菜名关键词
  - rn: 返回数量

### 4. 返回数据格式
```json
{
  "resultcode": "200",
  "reason": "Success",
  "result": {
    "data": [
      {
        "id": "1",
        "title": "宫保鸡丁",
        "ingredients": "鸡肉、胡萝卜、...",
        "burden": "油、盐、酱油...",
        "albums": ["图片URL"],
        "steps": [
          {"step": "1", "img": "步骤图片", "description": "步骤描述"}
        ]
      }
    ]
  }
}
```

## 方案二：百度AI开放平台

### 1. 注册账户

- 使用百度账号登录
- 实名认证

### 2. 创建应用
- 进入 "控制台" -> "应用列表"
- 创建新应用，选择 "图像识别" 或 "NLP"

### 3. 获取API信息
- AppID, API Key, Secret Key

### 4. 使用菜谱识别API
百度提供食材识别和菜谱推荐功能
